Programa de Estacionamento de Carros 
....................................

Este é um programa desenvolvido para facilitar a gestão de estacionamento de carros, oferecendo funcionalidades tanto para os usuários quanto para os administradores.


...::: Funcionalidades para Usuários :::...

Login: Os usuários devem-se autenticar para aceder aos parques e demais funcionalidades do programa.

Validação de Documentos: É necessário possuir uma carta de condução e um cartão bancário válidos para utilizar o parque.

Compra de Avenças: Os usuários têm a opção de comprar avenças para garantir estacionamento num lugar à sua escolha, pelo período de 1, 3, 6 ou 12 meses. Não há limite para o número de avenças.

Estacionamento único: O usuário pode entrar no parque e escolher um lugar disponível, independentemente de já ter uma ou mais avenças adquiridas.

Recarga de Saldo: Os usuários devem ter saldo positivo para poderem adquirir avenças e efetuarem estacionamentos individuais/ únicos.

Introdução de Informações do Veículo: Os usuários devem inserir as informações solicitadas sobre os seus veículos - matrícula e se é carro ou mota.


...::: Funcionalidades para administradores :::...

Gestão de Parques: Os administradores podem criar e editar informações sobre os parques de estacionamento, tais como:
- nomear os parques
- definir número de pisos e planta de cada um
- criar, editar e atribuir lugares de estacionamento em cada andar, incluindo se se destinam a carros ou motas.


...::: Como utilizar :::...

Requisitos de Sistema: Microsoft Visual Studio ou outro compatível com projetos ASP.NET

Instalação: Abra o projeto ParkNet e corra o programa.

Login: Após registo, faça login com as suas credenciais de usuário ou administrador.

Notas Adicionais: Certifique-se de manter as suas informações de login, carta de condução e cartão bancário em segurança.

Este programa destina-se apenas para fins de demonstração, em âmbito académico.
